Features

Everything you need to find the right modpack, fast — without juggling filters and tags.

Natural Language Search

Type what you want in plain text. No filters, no tags — just describe your ideal modpack.

3-Phase AI Pipeline

AI parses your query, Modrinth API finds candidates, then AI ranks and explains the best results.

Live Streaming Results

Server-Sent Events stream each phase in real time, so you see progress as the search runs.

Smart Fallbacks

If too few results match, the search progressively broadens with alternate queries to find more.

Category Exclusion

Say "no magic" or "no tech" — excluded categories are filtered out of your results.

Multiple AI Providers

OpenAI, Anthropic, Ollama, OpenRouter, OpenCode Zen, or any OpenAI-compatible endpoint.

Local Database

Optional SQLite + FTS5 crawler caches modpacks locally for instant, full-text search.

Secure by Default

Binds to localhost, masks API keys, restricts CORS, validates URLs, and rate-limits requests.

Fast & Responsive

Dark theme, keyboard shortcuts (Ctrl+K, /, Esc), search history, and a UI for every device.

How it Works

Four steps, fully automated — from a plain sentence to ranked recommendations.

1

Describe

Type a natural language query describing the modpack you're looking for.

2

Parse

The AI extracts search parameters: keywords, loaders, versions, categories, and exclusions.

3

Search

Queries the local database first, then the Modrinth API — broadening automatically if needed.

4

Rank

The AI ranks results and explains why each modpack fits your request.

Quick Start

Up and running in under a minute. All you need is Node.js 18+ and one AI provider key.

Install & Run
git clone https://github.com/alnyx-dev/Find-My-Modpack.git
cd Find-My-Modpack
npm install
cp .env.example .env   # add your AI provider key
npm start

# optional: live-reload dev server
npm run dev
# optional: build a local modpack database
npm run crawl

Open http://localhost:3000 in your browser. Providers can also be configured from the in-app settings UI.

Prerequisites

  • Node.js 18 or later
  • One AI provider API key

Supported Providers

  • OpenAI
  • Anthropic
  • Ollama (local, no key needed)
  • OpenRouter
  • OpenCode Zen
  • Any OpenAI-compatible API
