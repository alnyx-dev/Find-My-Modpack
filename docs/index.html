Features

Natural Language Search

Type what you want in plain text. No filters, no tags — just describe your ideal modpack.

AI-Powered Pipeline

Three-phase process: AI parses your query, Modrinth API finds matches, AI ranks the best results.

Smart Fallbacks

If too few results, the system progressively broadens your search to find the best matches.

Multiple AI Providers

OpenAI, Anthropic, Ollama, OpenRouter, OpenCode Zen, or any OpenAI-compatible endpoint.

AI Explanations

Each result comes with a personalized explanation of why it matches your request.

Fast & Responsive

Dark theme, keyboard shortcuts, and a responsive UI that works on any device.

Live Streaming Progress

Watch the AI work in real time with animated phase indicators and progress tracking.

Local Database

Optional SQLite crawler for faster local search with full-text search support.

How it Works

Describe

Type a natural language query describing the modpack you're looking for.

Parse

The AI extracts search parameters: keywords, loaders, versions, categories.

Search

Structured query searches the local database first, then Modrinth API. Results broaden if needed.

Rank

The AI ranks results and explains why each modpack fits your request.

Quick Start

Install & Run
git clone https://github.com/alnyx-dev/Find-My-Modpack.git
cd Find-My-Modpack
npm install
cp .env.example .env
npm start

Open http://localhost:3000 in your browser. Optionally run npm run crawl to populate a local modpack database for faster search.

Prerequisites

  • Node.js 18 or later
  • One AI provider API key

Supported Providers

  • OpenAI
  • Anthropic
  • Ollama (local, no key needed)
  • OpenRouter
  • OpenCode Zen
  • Any OpenAI-compatible API
